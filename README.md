## Smart contracts

### AzboxDeposit.sol

`AzboxDeposit.sol` is Azbox's deposit contract, allowing users to directly deposit funds through this contract.

### AzboxWithdrawal.sol

`AzboxWithdrawal.sol` is the withdrawal contract for Azbox.
